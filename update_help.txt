Usage: insforge secrets update [options] <key>

Update an existing secret

Options:
  --value <value>    New secret value
  --active <bool>    Set active status (true/false)
  --reserved <bool>  Set reserved status (true/false)
  --expires <date>   Expiration date (ISO 8601, or "null" to remove)
  -h, --help         display help for command
